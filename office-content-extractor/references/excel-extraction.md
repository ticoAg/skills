# Excel 内容提取

## 目录

- 输入防御
- sheet 级提取
- 表格输出
- 通用文本输出

## 输入防御

Excel 常见问题：

- 脏 `dimension`
- `max_row/max_column` 被污染
- 远处孤立单元格
- 超大空表

提取前优先：

- 校验可见 sheet 数量
- 校验行列上限
- 必要时预清洗 OOXML

## sheet 级提取

把 sheet 当作一级内容边界。

至少保留：

- `sheet_name`
- sheet 顺序
- 行内单元格顺序

不要默认隐藏 sheet 也要参与提取。

## 表格输出

当数据天然是结构化表格时，优先：

- 保留列边界
- 适度保留表头
- 输出 Markdown 表格或行级键值文本

如果表格特别宽，不要生硬输出巨型 Markdown 表；可以退化成“每行非空单元格拼接”的紧凑文本。

## 通用文本输出

如果只是为了提内容：

- 按行拼接非空单元格
- 用制表符或明确分隔符保留弱结构
- 再做统一分块

这通常比整表原样渲染更稳。
